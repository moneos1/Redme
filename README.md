# Moneos Network

Bienvenue sur le site officiel de **Moneos Network** !

## Présentation

Moneos Network est un écosystème innovant dans la finance décentralisée (DeFi) et le Web3. Notre objectif : offrir une expérience simple, sécurisée et transparente autour de la crypto-monnaie et de la blockchain, notamment sur la Binance Smart Chain (BSC).

## Fonctions principales

- **Staking avantageux** : Profitez d’un rendement de 10% sur 28 jours grâce à notre smart contract sur la BSC.
- **Smart Contracts** : Sécurité et transparence garanties par la technologie blockchain.
- **Dashboard Web3** : Connectez facilement votre portefeuille Metamask pour accéder à toutes les fonctionnalités.
- **Parrainage** : Un système de parrainage simple via un lien unique pour récompenser la communauté.
- **DAO participative** : Participez aux décisions importantes via notre organisation autonome décentralisée.
